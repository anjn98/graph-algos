GENERAL FORMAT


'path'

It should a string representing path. It should have have edge list representation where nodes can be either numerical or string(without spaces) and every 3rd word should be an intger representing the edge weight between previous two nodes

'start'

It represnts the start point. It can be either numerical or string(without spaces)

'end'

It represnts the end point. It can be either numerical or string(without spaces)



OUTPUT

CASE 1: IF THERE IS SOME ERROR

Ouput will have only one variable named Error which will be set to true


CASE 2: IF THERE IS NO ERROR AND PATH NOT FOUND

Ouput will have only two variable, Error which will be set to false and another Found also set to false


CASE 3: IF THERE IS NO ERROR AND PATH FOUND

Error set to false Found set to true.
There will be additional parameters
Length, the shortest path length
Start which has start point
End which has end point
Path a list which shows the shortest path
